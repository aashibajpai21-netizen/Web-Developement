📌 Project Description

This project is a multi-page website built using only HTML. It showcases different HTML tasks and assignments, demonstrating the use of basic HTML elements like forms, tables, lists, links, and more.

The website contains a main homepage and multiple task pages connected through navigation links.

🎯 Objective
To understand the structure of a website using pure HTML
To practice creating multiple linked web pages
To demonstrate different HTML concepts in separate task pages


🛠️ Technologies Used
HTML5 (Only)
No CSS
No JavaScript

🔗 Features
🏠 Homepage with navigation links
📄 Multiple task pages (Task 1 to Task 16)
🔙 “Back to Home” link on every page
🧾 Forms (Registration, Feedback, etc.)
📊 Tables and Lists
🔗 Internal linking using anchor tags
▶️ How to Run the Project
Download or clone the repository
Open the project folder
Double-click on index.html
The website will open in your browser
